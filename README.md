# BUDDICA CRM Project

カーライフアドバイザー向けCRMシステムの開発プロジェクト

## プロジェクト概要
- **目的**: 成約率向上、接客時間短縮、データ電子化
- **技術**: Supabase + Vercel + BigQuery連携
- **対象**: iPad使用のカーライフアドバイザー

## ディレクトリ構造
- `docs/`: 企画・設計ドキュメント
- `prototype/`: プロトタイプ・モックアップ
- `src/`: ソースコード（今後追加）

## 主要機能
1. 顧客管理・商談パイプライン
2. 在庫検索・他社比較
3. 電子契約・見積もり作成
4. チーム連携・店舗間情報共有
5. LINE/メール統合コミュニケーション

## 開発フェーズ
- Phase 1: MVP（基本機能）
- Phase 2: 拡張機能（チーム連携・分析）  
- Phase 3: 最適化（AI活用・自動化）
